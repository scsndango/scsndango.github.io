Java 小知识

这份博客是平时遇到的java小知识的一个总结，也相当于占坑

  1. java 程序的编译运行原理
  2. java包管理软件 maven的简单介绍和使用
  3. java jre 和 jdk的区别
    1. jre (java runtime environment) 是一个java运行时环境，相当于可以运行编译好的java程序（符合java多平台传播的理念，我感觉就是把java虚拟机包括了进去）
    2. jdk(java SE development kits) 是开发java程序使用的一些工具、软件的集成器，只有安装了jdk才能编译java程序
