周志华-机器学习-第一章 绪论

发表于 2019-04-05 | 分类于 机器学习-西瓜书阅读

1.1 引言

机器学习概念

周志华的定义： 机器学习致力于研究如何通过计算的手段，利用经验来改善系统自身的性能。

Mitchell 1997更为形式化的定义： 假设用$P$ 来评估计算机程序在某任务类$T$上的性能，若一个程序通过利用经验$E$在$T$中任务上获得了性能改善，则我们就说关于$T$和$P$，该程序对$E$进行了学习。

机器学习研究的主要内容是学习算法(learning algorithm)。

1.2 基本术语

数据相关的一些概念：

示例(instance)/样本(sample):关于一个事件或对象的一条记录。比如 关系抽取任务（Relation Extraction）中的一个sentence bag；MNIST 手写数字识别任务中的一张28x28的图片。

属性(attribute)/特征(feature)：反映事件或者对象在某方面的表现或性质的事项。比如MNIST中的每个像素。

属性空间(attribute space)/样本空间(sample space)：属性张成的空间。

一个示例的所有属性构成的向量称为特征向量(feature vector).

训练过程相关的概念：

学习(learning)/训练(training)：从数据中学得模型的过程。

训练数据(train data)：训练过程中使用的数据

训练样本(train sample)：训练过程中使用的一个样本，所有的样本组成训练集(train dataset)

学习到的模型对应了关于数据的某种假设(hypothesis)，数据潜在的规律自身被称为真相或者真实(ground-truth)

测试过程相关的概念：

测试(testing)：利用学习到的模型对新样本进行预测的过程

被预测的样本称为 测试样本(test sample)，所有测试样本组成测试集(test dataset)

任务信息相关的概念：

学习任务可大致划分为两大类：监督学习(supervised learning) (例如：分类 classification 和回归 regression 任务) 和 无监督学习(unsupervised learning) (例如：聚类 clustering 任务).

机器学习目标：

泛化(generalization)能力：学得模型适用于新样本的能力。

机器学习的目标就是学习具有强泛化能力的模型。

1.3 假设空间

假设空间 ：监督学习的目的在于学习一个由输入到输出的映射，这一映射由模型来表示。话句话说，学习的目的就在于找到最好的这样的模型。模型属于有输入空间到输出空间的映射集合，这个集合就是假设空间(hypothesis space)。假设空间的确定意味着学习范围的确定。

李航 《统计学习方法》

The hypothesis space, which defines the class of functions mapping the input space to the output space. That is, the functions operate on the feature vectors of the input objects, and make predictions according to the format of the output space.

刘铁岩 《Learning to Rank for Information Retrieval》

版本空间：假设空间中与训练集一直的假设组成的集合。

1.4 归纳偏好

归纳偏好 可看作学习算法自身在一个可能很庞大的假设空间中对假设进行选择的启发式或“价值观”。

奥卡姆剃刀(Occam’s razor)：若有多个假设与观察一致，则选择最简单的那个。

当假设空间中存在多个假设与测试集一致的时候我们依据奥卡姆剃刀的归纳偏好倾向于选择更加简单平滑的A曲线。
